# 🌡️ Temperature Converter

A simple Python program to convert temperature between **Celsius** and **Fahrenheit**.

##   Description
This is a beginner-friendly Python script that takes a temperature value and converts it 
from Celsius to Fahrenheit or vice versa, based on user input.

##  Installation

### Prerequisites
- Python 3.6 or higher installed on your system

### Step 1: Check if Python is installed
Open your terminal or command prompt and run:
```bash
python --version
```
If Python is not installed, download it from [python.org](https://www.python.org/downloads/) 
and make sure to check **"Add Python to PATH"** during installation.

### Step 2: Clone the Repository
```bash
git clone https://github.com/maryam-hamid-sa/Temperature-Converter.git
```

### Step 3: Navigate to the Project Folder
```bash
cd Temperature-Converter
```

### Step 4: Run the Program
```bash
python "temperature convertor.py"
```

##  Example

Enter temperature value: 100

Convert to Celsius or Fahrenheit(C/F): F

Temperature in Fahrenheit: 212.0 F
