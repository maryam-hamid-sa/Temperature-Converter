# 🌡️ Temperature Converter

A simple Python program to convert temperature between **Celsius** and **Fahrenheit**.

##  Description
This is a beginner-friendly Python script that takes a temperature value and converts it 
from Celsius to Fahrenheit or vice versa, based on user input.

##  How to Run
1. Make sure Python is installed on your system
2. Clone this repository or download the file
3. Run the script:
```bash
   python "temperature convertor.py"
```
4. Enter the temperature value
5. Enter `C` to convert to Celsius or `F` to convert to Fahrenheit

##  Example
Enter temperature value: 100
Convert to Celsius or Fahrenheit(C/F): F
Temperature in Fahrenheit: 212.0 F
